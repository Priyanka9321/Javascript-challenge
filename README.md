## How to Use This Repository

1. Clone the repository:
    ```bash
    git clone https://github.com/Priyanka9321/Javascript-challenge.git
    ```

2. Navigate to the day's folder and explore the solutions:
    ```bash
    cd Javascript-challenge/Day1
    ```

3. Run the code:
    ```bash
    node index.js
    ```
